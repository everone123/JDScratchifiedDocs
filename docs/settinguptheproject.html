3 - Setting Up the Project

We've prepared everything we need, so we're finally ready to open up the .sb3 project you downloaded earlier.

Warning

Use the TurboWarp editor. You WILL NOT be able to follow this guide using only vanilla Scratch.

If you want to be able to access your project from other devices, save it to a flash drive or use a cloud storage service like Google Drive.

You will still be able to share your project to Scratch once you're finished.

Song Metadata​

The JDS engine contains several lists (pictured below) that define important metadata for songs. The names of the lists all start with "Song" so that they're conveniently placed next to each other in the project's... uh, list of lists.

These lists all contain one item per song. If you're using the blank JDS template project (which you should be, unless you're trying to add multiple custom songs to one project), then your song's index in these lists will be 1. That means the first item in each list will be the one that your routine uses.

In the JDS template project, the lists already contain one "example" item, which you should now replace as follows:

List NameWhat you should enter
Song # of CoachesThe number of coaches in your routine (1 for Solos, 2 for Duets, and so on). If you're using non-JD media, any dancers with exactly the same choreography should be counted as 1 coach.
Song BPMsThe tempo of the song you're using, in beats per minute. You can use decimals if you need to. Mid-song tempo changes are unfortunately not yet supported, but this may change in a future update.
Song CodenamesThe same codename that you used as the file name for your GIF. This is CASE SENSITIVE. Do not include the .gif extension.
Song DelaysLeave this at 0 for now. We'll figure it out later.
Song ListThe actual title of your routine. Keep it under 40 characters if you can.
Song Lyric ColorsThe color that will be applied to words in the lyrics when they are highlighted. This works like a hue shift, with a range of 0 to 200, using red as the base.
Song Origin GamesThe Just Dance game that your routine is from. This uses abbreviations like "JD1", "JD2024", and so on. Enter "FANMADE" for fan-made routines, or "FANMADEMU" for fan-made Mashups. Enter "NA" if your routine isn't from JD.
Song VolumesLeave this at 75 for now. Once you import the song to the project, you can adjust the audio balancing at any time by changing this value. Caps at 100.
Tip

JDS works best with BPMs between 80 and 180. If your song's tempo is not within this range, consider halving or doubling the value in the Song BPMs list to make it fit.

Menu Setup​

We need to actually be able to select the song in-game to enter it, so let's do a bit of preparation for that.

  1. Open the "Menus" sprite folder.
  2. Click on the "Menu Icons" sprite to select it.
  3. Open the Costumes tab.
Warning

Many sprites in the game (including this one) have a "boundary test" costume consisting of a large white rectangle. Never mess with it- the game WILL break if you do.

  1. Select the costume called "Song Name".
  2. Replace that costume's name with your song's title (it must be exactly the same as in the Song List).
  3. Change the text at the bottom of the costume itself to match your song's title. You can use multiple lines if you need to.
  4. With the Menus sprite folder still open, select the "Highlighted Title" sprite.
  5. Select the "Codename" costume and replace its name with the song codename you've been using.
  6. Change the costume's "Artist" and "Song Title" text boxes as appropriate for your routine. You can resize them and/or use multiple lines if you need to.
  7. Go back to the "Menus" sprite folder once again and select the "Coach + Text" sprite.
  8. There are placeholder costumes called "1Codename", "2Codename" and so on. Replace "Codename" in these costume names with your routine's actual codename.

Your song should now appear on the menu. It will act a bit glitchy when highlighted because no preview has been made yet, but we'll deal with that later on, so don't worry about it yet.

Menu Visuals​

If you'd like to go ahead and do this now, you can also add the visuals for the menu icon and coach select portraits. You can get these from a song's page on the Just Dance wiki (yes, Fandom is disgusting, but there is no alternative here), or you can remake them yourself.

If your routine isn't from JD, you'll need to get a bit creative with the menu icon and coach portrait(s), but it shouldn't be too difficult. There are examples provided in the project.

If you'd prefer to wait before adding these visuals (I usually add them after I've actually made the routine so I can use a screenshot as the menu icon), you can just come back here later.

Importing Assets​

Now, we'll actually upload the files we got earlier. Do the following in order:

  1. Upload your song's .mp3 into the Backdrop. Then, change the sound's name to the name you entered into the Song List. Like before, it must match exactly.
  2. Save a backup copy of the project before continuing. If you need to undo the GIF upload for any reason, reloading this save is easier than manually deleting all the costumes.
  3. Locate the sprite called "[Song Title Here]" (it's near the bottom). We will insert the GIF of your routine's coach(es) into this sprite. Replace the sprite name with the name of your song. Then, go to the Costumes tab and upload your GIF. It will take a minute or two; let it do its thing until the loading icon at the top disappears.

Now that your GIF is in the project, we need to make it actually "work" in-game. This will be done on the next page.
