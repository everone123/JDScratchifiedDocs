7 - Adding Lyrics

Now, we'll add on-screen lyrics to your routine. They will appear during "real" gameplay (i.e. not test runs) as long as Low-Resource Mode is turned off.

The lyrics are purely visual. If your song doesn't have any vocals, or you don't want to put in the extra work to add them to your routine, you can skip this page.

Caution

JDS lyrics currently only support English alphanumerals and the following special characters: . , ! ? - ' "

Support for custom lyric characters will be added in a future version.

Writing the Lyrics​

Do this in Notepad or another text editing software. We will import it into a list in the project once you're done.

Each line of the lyrics should be no more than a few words. Avoid going over 40 characters per line.

Note

This per-line limit is suggested for two reasons: to avoid overflowing into the pictograms area, and to make sure the game has enough time to load each line (it can generate up to 30 characters per second, excluding spaces).

You may have noticed the use of underscores in the image example above. When writing lyrics, you use underscores to separate syllables that will be highlighted separately. The underscores will not be rendered in-game; they just tell the lyric generator what to do when highlighting words.

Also, use an asterisk "*" in place of any censored words. It will appear as "..." in-game.

Setting the Timings​

This is where we (finally) boot up ArrowVortex, which you should have downloaded in part 1 if you were planning to add lyrics to your routine.

Disclaimer

Neither Just Dance Scratchified nor its developer have any official affiliation with ArrowVortex. This tool was not specifically designed for JDS, but we are using it because it incidentally allows this process to be much more simple and efficient.

When you boot up ArrowVortex, you'll be greeted by a blank screen with their logo. Press Ctrl+O (or select File > Open from the menu bar at the top). Then, choose the MP3 that you've been using in-game.

Next, press Shift+N (or select Chart > New chart...). Set the chart type to Dance Double. The other properties don't matter. Press the confirm buttom and create the chart.

Bring up the sync menu with Shift + S (or Tempo > Adjust sync). Enter the tempo into the "Initial BPM" and click Apply BPM.

Charting​

ArrowVortex is a rhythm game charting software. Since we selected Dance Double as our chart type, you should see 8 arrows at the top of the screen. Before we get into how we'll be using these, let's go over the basic controls:

  • Press the up and down arrows to scroll up and down.
    • Press left and right to change the snap range.
  • Press the spacebar to play and pause.
  • Press + or - on your numberpad to zoom. (You can also use View > Zoom > Zoom in/out.)
  • Press any number key 1 through 8 to place an arrow in that column. We will only be using columns 1 through 6.
    • You can delete an arrow by pressing the same key again in the same spot.

Go ahead and set your snap range to "8th" (so the diamonds next to the arrows are blue). For JDS, we will never need to use a snap more precise than this.

Each arrow that you place on the chart will correspond to a certain event in your routine's lyrics. Here are the events attributed to each column:

ColumnFunction
1Each arrow will highlight one word (or syllable) in the lyrics. Words are always highlighted in order, and the game will not automatically catch up if a line disappears before it is fully highlighted.
2Same as 1. Use if you need to highlight 2 words within the same half-beat.
3Each arrow will cause the a lyric line to appear. They always appear in order. Remember that they need enough time to load.
4Each arrow will cause the current lyric line to disappear.
5Each arrow will cause a backing lyric line to appear. This is an advanced feature. You can enter backing lines in the Costumes tab of the game's "Backing Lyrics" sprite, and you can set the order via the "backingOrder" variable setter near the top of the coach sprite's main script.
6Each arrow will cause the current backing lyric line to disappear.
7Unused.
8Unused.

Converting​

Once your "chart" is finished, we'll run it through a Scratch project that converts it to a format readable by JDS. The project is pretty small and we only need to use it once, so it's easiest to just open it in a browser. Here you go!

Right-click on the "ArrowVortex Data" list and import your ArrowVortex .sm file. Next, import your written lyrics into the "Lyrics" list. Then, run the "Convert ArrowVortex to JDS Lyrics" custom block, and export the Output list to your computer.

Finally, go back to your custom song project, and scroll up to the top of the Lists section. You should see six lists that all start with "_Codename". These are the song-specific data lists. To keep things organized, you can go ahead and replace "Codename" in these list names with your song's codename (don't delete the underscore at the beginning). If your song has fewer than 4 coaches, you can also delete any Judge Data lists that you don't need.

Import the Output file that you just got into the "Lyric Data" list. Lyrics should now be working properly in-game.
